## üßê About <a name = "about"></a>

Collabora App is a new project which aim is to create a community and plataform where people and companies can access and share resources they don't need. The purpose is to collaborate with Circular Economy by re-using resources.

## üèÅ Getting Started <a name = "getting_started"></a>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See [deployment](#deployment) for notes on how to deploy the project on a live system.

### Prerequisites

For being able to start you need all those requirements:
- NodeJS
- NPM

Once the requirements are satisfied you can continue with the following commands:
```
npm install @ionic/cli cordova-res
cd ~/collabora
npm install && ionic serve
```
